# 🔐 Password Security Checker

A simple web app that shows how long it would take to crack a password.

## 🛠 Tech Stack

* Python (Flask)
* HTML / CSS / JavaScript

## ▶️ How to Run

```bash
pip install flask
python app.py
```

Open in browser:

```
http://127.0.0.1:5000
```

## ⚠️ Disclaimer

This tool is for educational purposes only.
Do not enter real passwords.
